# Your Platform

**Описание:**  
Простое приложение на PyQt5 с поддержкой переключения тем (светлая/тёмная).  
Позволяет динамически менять стиль интерфейса.

## Структура проекта
```
your_platform/          ← Корневая папка проекта
│   main.py             ← Файл в корне
│   README.md           ← Файл в корне
│
├───resources           ← Подпапка
│   └───styles
│       ├───dark
│       │       main.qss
│       └───light
│               main.qss
│
└───utils               ← Подпапка
        theme_manager.py
        __init__.py
```
