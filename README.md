# Your Platform

**Описание:**  
Простое приложение на PyQt5 с поддержкой переключения тем (светлая/тёмная).  
Позволяет динамически менять стиль интерфейса.

## Структура проекта
```
your_platform/
│   main.py
│
├───resources
│   └───styles
│       ├───dark
│       │       main.qss
│       │
│       └───light
│               main.qss
│
└───utils
    │   theme_manager.py
    │   __init__.py
```
